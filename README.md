# 🌕 Artemis II – Wikipedia Page Clone

-----

## 📌 Project Overview

This project is a simplified Wikipedia-style article page dedicated to **Artemis II**, NASA’s first crewed mission under the Artemis program.
It presents the mission’s crew, history, mission profile, spacecraft details, and significance in a clean, structured, and informative layout that closely mirrors the design of a real Wikipedia article page.

The project demonstrates the use of HTML and CSS to build a well-organised, multi-section reference page with navigation, a table of contents, superscript citations, and a references list.

-----

## 🎯 Problem Statement

Finding structured, readable information about space missions can be overwhelming across multiple sources.
This Wikipedia-style page brings all key information about the Artemis II mission into one clean, easy-to-navigate digital document.

-----

## ✨ Features

- Wikipedia-style top navigation bar with search input
- Page tabs (Article, Talk, Read, Edit, View history)
- Left sidebar with navigation, tools, and language links
- Infobox summarising key mission details
- Table of contents with jump links to all sections
- Superscript citation numbers linking to the references section
- Sections covering Introduction, Crew, History, Mission Profile, Spacecraft, and Significance
- Ordered references list with links to real sources
- Edit links on every section heading
- Fully responsive design for mobile, tablet, and desktop

-----

## 📄 Pages Included

- **Wikipedia Article Page (index.html)** – Single page with all article content including infobox, table of contents, sections, citations, and references

-----

## 🛠️ Technologies Used

- **HTML** – Semantic page structure, tables, lists, superscripts, and iframes
- **CSS** – Wikipedia-inspired styling, layout, and responsive design using media queries

## 🚀 How to Run the Project

1. Download or clone the repository
1. Open the project folder in **VS Code**
1. Right-click `index.html` and select **Open with Live Server**
1. The Wikipedia article page opens in your browser
1. Click the table of contents links to jump to sections
1. Click the superscript numbers to jump to the references

-----

## 🌐 How to Host on GitHub Pages

1. Create a new GitHub repository named `wikipedia-page`
1. Upload `index.html` and `README.md`
1. Go to **Settings → Pages**
1. Under **Branch**, select `main` → click **Save**
1. Your live site will be at:
`https://ndungupatriciawanjiru-hub0.github.io/wikipedia-page`

## ⚠️ Known Limitations

- Single page only — no real search functionality
- Links in the sidebar and tabs do not navigate to real pages
- No JavaScript interactivity
- Content is static and does not update automatically
